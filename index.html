Qual é o Nosso Objetivo?

O objetivo do projeto Gabaritei o Enem tem como objetivo preparar jovens e até adultos para fazer exatamente o que o nome do projeto diz, gabaritar o Enem. Esse projeto é feito por um estudante de ensino médio, porque pelas palavras do criador:

"Um estudante consegue compreender as dificuldades de outro estudante por estar em uma mesma situação"

Pois é baseado nessa citação do criador que este site se baseia, porque uma pessoa que não é mais estudante pode não lembrar-se mais das suas dificuldades quando estudante, por exemplo um professor. Entretanto um estudante pode compreender as dificuldades de outro estudante, este é um resumo do que pretendemos fazer, ajudar pessoas a passar no Enem!
